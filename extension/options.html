How to Use

Your tiddlywikis must be location in a directory called 'tiddlywikilocations' that must be located
In the download directory.

And

The browser option "Ask where to save each file" MUST be switched off.
In chrome this is in the advanced options in the 'Downloads' section.
In firefox this is in the prefrences, General, Files and Applications

Enable for a once a day backup. Date is appended
to the name of the Tiddlywiki.
